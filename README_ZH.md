# ⚡ AI Skill Hub Manager (v1.0.0 正式版)

> 🚀 **零门槛、零依赖的 Windows 原生独立双端 AI 协同开发规约与本地技能库可视化管理器。**

**AI Skill Hub Manager** 是一款专为 AI 时代协同开发量身打造的 Windows 桌面客户端。本系统能够跨项目、跨框架高效管理、编辑您的 AI 技能约束文件（如 `.md` 开发规约与红线准则），并支持一键将选定的技能装载分发到您指定的任意开发项目中，自动在目标项目内构建高效的哈希校验与 `AGENTS.md` 索引目录。

## 🎨 核心特性

- 📦 **100% 独立单文件运行 (.exe)**：
  - 彻底摆脱“本地 Web 服务监听 + 浏览器套壳”的易冲突架构。基于 `pywebview` 引擎直接读取本地静态资源包，不监听任何端口，零网络依赖。
  - 彻底去除 CMD 黑窗口闪烁，双击即可秒级拉起原生清爽窗口。
- ⚙️ **自适应路径系统 (v1.0.0 核心突破)**：
  - **路径完全去硬编码**：首次启动程序会自动在同级目录下新建默认的 `skills/` 全局技能库文件夹，实现**即开即用、零配置开箱体验**！
  - **原生目录选取器**：支持在 UI 中直观地一键切换、选取自定义全局技能库（D 盘、C 盘、用户文档目录等），彻底适配各种系统差异与网络分发。
- 📂 **智能项目选取与状态哈希审计**：
  - 点击“关联开发项目”一键唤起 Windows 标准原生文件夹选择框，0 键盘输入，极致防错。
  - 精准进行项目备份与全局库文件哈希审计，智能呈现状态图标：
    - 🟢 **已同步 (Synced)**：项目端规约与全局库 MD5 校验绝对匹配。
    - 🟡 **有更新 (Out of sync)**：检测到全局技能指南被更新，提示一键物理分发同步。
    - ⚪ **未装载 (Unloaded)**：项目目录中暂无此技能规约。
- 📝 **内置等宽高级 Markdown 原生模态编辑器**：
  - 点击“编辑技能”直接唤起自带编辑框，模态置顶，光标秒级自动聚焦，支持实时渲染预览。
- 🎨 **极客微动交互**：
  - Stripe 风格双行数据看板、iOS 自适应滑块、自适应弹性伸缩高度的卡片排版。
  - 鼠标悬浮项目及卡片时，触发科技感动态呼吸边框微光特效（Border Glow Effect）。

## 💻 安装说明 (Installation Guide)

### 方式 A：直接运行成品 (推荐)
1. 前往 GitHub 的 **Releases** 页面下载最新版的 `AI_Skill_Hub_Manager.exe`。
2. 将 `.exe` 放置在您电脑上的任意文件夹（如 `D:\DevApps\skill_store`）。
3. 双击 `AI_Skill_Hub_Manager.exe` 即可启动运行。
4. *注意：首次运行会在同级目录下自动生成 `config.json` 配置文件与 `skills/` 默认全局技能库文件夹。*

### 方式 B：从源码本地安装运行
如果您想在本地运行源码或进行二次开发，请按以下步骤操作：

#### 1. 克隆仓库
```bash
git clone https://github.com/您的用户名/您的仓库名.git
cd skill_store
```

#### 2. 安装 Python 环境
确保您的 Windows 系统已安装 Python 3.10 或以上版本（推荐使用标准 Python 环境）。

#### 3. 安装第三方依赖库
本程序基于 Python 的 `pywebview` 库实现原生窗口绑定，运行以下命令安装所需依赖：
```bash
pip install pywebview
```

#### 4. 启动程序
```bash
python main.py
```

## 🛠️ 二次打包与编译指南 (Compilation Guide)

如果您修改了代码（特别是 `static/` 下的 HTML/CSS/JS 资产），需要将其重新打包编译为独立的单文件 `.exe` 时，请按如下步骤操作：

### 1. 安装打包工具
```bash
pip install pyinstaller
```

### 2. 执行一键编译打包
使用 PyInstaller 将所有前端资产及依赖完整压缩封装为一个独立的 EXE 运行包：
```bash
pyinstaller --noconsole --onefile --clean --add-data "static;static" main.py
```

### 3. 获取成品
编译完成后，可在生成的 `dist/` 目录下找到 `main.exe`。将其重命名为 `AI_Skill_Hub_Manager.exe` 并放置在项目根目录即可分发给他人使用。

## 🕹️ 详细使用说明 (Usage Manual)

### 第一步：设置全局技能库
1. 打开软件后，在左侧侧边栏的 **“全局技能库”** 卡片中，可以看到当前的加载路径。
2. 如果您希望将其切换为您已有的规约目录（如 `D:\DevApps\skills`），点击卡片右侧的 ⚙️ 设置按钮，在弹出的系统对话框中选中目标文件夹即可。
3. 您可以点击左侧侧边栏上方的 **“新建技能”** 按钮，输入文件名（如 `代码移交标准.md`），系统会自动生成精美的 Markdown 模板并为您拉起编辑器。

### 第二步：关联您的开发项目
1. 点击左侧侧边栏上方的 **“关联项目”** 按钮。
2. 在弹出的原生文件夹选取框中，选择您正在开发的任何项目根目录（例如 `D:\Project\my-app`）。
3. 关联成功后，左侧项目列表中会显示该项目。点击选中它，右侧会即时刷新该项目相对于全局技能库的装载状态。

### 第三步：一键同步与自动索引
1. 在右侧全局技能列表中，通过每个卡片右下角的 **“启用装载”** 开关，勾选当前项目需要启用的技能。
2. 勾选完毕后，右上角的 **“一键同步技能”** 按钮将亮起。
3. 点击 **“一键同步技能”** 按钮，系统会自动：
   - 将选中的 `.md` 技能规约文件拷贝分发到您项目的 `.agent/skills/` 目录下。
   - 自动在您项目的根目录下生成或更新一份排版精美的全中文 [**`AGENTS.md`**](AGENTS.md) 导航索引文件，供 Cursor、GitHub Copilot 等 AI 工具一键读取，并完美实现规约生效！
   - 自动卸载并物理删除项目目录中您未勾选的规约文件，保持项目绝对纯净。
