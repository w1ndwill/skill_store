# Superpowers Task Checklist

Use this file to track your active tasks. Each step should be highly granular (2-5 minutes).

## 🚀 Active Checklist

- [ ] **Phase 1: Brainstorming**
  - [ ] Explore codebase and define the scope
  - [ ] Map out edge cases and tech constraints
  - [ ] Align with user on chosen architectural design
- [ ] **Phase 2: Planning**
  - [ ] Write implementation plan in `docs/plans/implementation_plan.md`
  - [ ] Detail verification criteria and tests
  - [ ] Obtain explicit user approval to proceed
- [ ] **Phase 3: TDD Execution**
  - [ ] Write/update unit tests first (Red stage)
  - [ ] Implement minimum code changes to pass (Green stage)
  - [ ] Refactor and verify clean code
- [ ] **Phase 4: Verification & Walkthrough**
  - [ ] Run full automated test suite
  - [ ] Complete code review checklist
  - [ ] Generate final `docs/plans/walkthrough.md`
